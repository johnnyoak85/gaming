# Sega Master System Mini

List of games. Fantasy console. Probably on Raspberry Pi 4.

| Game                                          | Why include it                                                                                                                                                                     |
| --------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Phantasy Star**                             | The system’s crown jewel. One of the few 8-bit RPGs that still feels historically important and genuinely ambitious, with sci-fi fantasy flavour instead of pure medieval fantasy. |
| **Wonder Boy III: The Dragon’s Trap**         | Probably the best “play it today” action-adventure on the console. The Master System’s transformation-based classic. Mandatory.                                                    |
| **Alex Kidd in Miracle World**                | Not flawless, but essential Sega identity. Before Sonic, this was Sega’s mascot platformer, and on Master System it is culturally unavoidable.                                     |
| **Shinobi**                                   | One of the best examples of Sega arcade DNA translated to the system. Cleaner, more deliberate, and more console-playable than expected.                                           |
| **Fantasy Zone**                              | Pure Sega arcade colour and charm. Important because it shows the Master System doing bright, weird, non-Nintendo energy extremely well.                                           |
| **R-Type**                                    | One of the showcase third-party arcade conversions. It has no business being as impressive as it is on 8-bit hardware.                                                             |
| **Castle of Illusion starring Mickey Mouse**  | Arguably one of the best licensed platformers on the system. Polished, charming, and very representative of late Master System quality.                                            |
| **Ninja Gaiden**                              | Not the NES game. This is its own Master System entry, and it is one of the strongest action games on the machine.                                                                 |
| **Power Strike / Aleste**                     | Essential shooter. Compile quality, fast, clean, and one of the system’s strongest “serious arcade skill” games.                                                                   |
| **Master of Darkness**                        | The closest thing to “Master System Castlevania,” and good enough to deserve the comparison rather than just living as a curiosity.                                                |
| **Wonder Boy in Monster Land**                | More arcade-RPG than Dragon’s Trap, but still very important. A little rougher, still very playable.                                                                               |
| **Wonder Boy in Monster World**               | Late, impressive, and worth owning if you care about the Wonder Boy lineage, even if the Mega Drive version is the main one.                                                       |
| **Sonic the Hedgehog**                        | Not the Mega Drive game. The Master System/Game Gear Sonic is a different 8-bit interpretation and absolutely worth treating separately.                                           |
| **Sonic the Hedgehog 2**                      | Also distinct from the 16-bit version. Harder, weirder, and very Master System/Game Gear-coded.                                                                                    |
| **Asterix**                                   | One of the best PAL-era platformers. Very polished, very European Master System energy.                                                                                            |
| **Asterix and the Secret Mission**            | Another excellent late-platformer pick, especially if you want the PAL personality of the console represented.                                                                     |
| **Golvellius: Valley of Doom**                | Sega’s Zelda-ish action-adventure. Not as elegant, but fascinating and substantial.                                                                                                |
| **Kenseiden**                                 | Samurai action-adventure with a darker tone. Not perfect, but distinctive and very worth knowing.                                                                                  |
| **Zillion**                                   | Another important Sega action-adventure. Slower and more password/maze-oriented, but part of the console’s identity.                                                               |
| **Golden Axe Warrior**                        | A shameless Zelda-like, yes. But a good one, and very desirable as a collector piece.                                                                                              |
| **OutRun**                                    | Iconic Sega arcade representation.                                                                                                                                                 |
| **Space Harrier**                             | Technically compromised, but historically important.                                                                                                                               |
| **After Burner**                              | Not ideal, but very Sega. Important arcade lineage.                                                                                                                                |
| **Hang-On**                                   | Foundational early Sega racing.                                                                                                                                                    |
| **Enduro Racer**                              | More enjoyable than you might expect, and a good arcade-style racing pick.                                                                                                         |
| **Alien Syndrome**                            | Good arcade horror-action flavour.                                                                                                                                                 |
| **The Ninja**                                 | Early and simple, but has that clean Sega arcade feel.                                                                                                                             |
| **Choplifter**                                | Solid arcade-style action.                                                                                                                                                         |
| **Quartet**                                   | More interesting historically than essential, but worth including for arcade texture.                                                                                              |
| **Psycho Fox**                                | One of the console’s personality pieces, with a distinct Master System feel.                                                                                                       |
| **Power Strike II**                           | One of the premium late shooters and very desirable. Expensive, but extremely curator-worthy.                                                                                      |
| **The Lucky Dime Caper starring Donald Duck** | Excellent Disney platformer; pairs nicely with Castle of Illusion.                                                                                                                 |
| **Land of Illusion starring Mickey Mouse**    | Another polished Disney entry, with more late-era Master System flavour.                                                                                                           |
| **Deep Duck Trouble starring Donald Duck**    | Late, attractive, and very PAL/Brazil Master System energy.                                                                                                                        |
| **Bubble Bobble**                             | Great single-screen arcade comfort food; the Master System version is an easy recommendation.                                                                                      |
| **Rainbow Islands**                           | Strong arcade conversion, cheerful, and still very playable.                                                                                                                       |
| **Bank Panic**                                | Simple but excellent. One of those “why is this so addictive?” arcade-style games.                                                                                                 |
| **SpellCaster**                               | Weird action/adventure/visual-novel hybrid. Very Sega, very odd, very worth knowing.                                                                                               |
| **Lord of the Sword**                         | Rough, but distinctive. More historical texture than pure recommendation.                                                                                                          |
| **Penguin Land**                              | Charming puzzle-platformer and a good softer counterweight to the arcade/action-heavy library.                                                                                     |
| **Sonic Chaos**                               | The third 8-bit Sonic and the first one that really feels more relaxed, colorful, and “portable Sonic” rather than a weird alternate take on Mega Drive Sonic.                     |

Total: 41 games
