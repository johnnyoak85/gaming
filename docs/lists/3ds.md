# Nintendo 3DS

## Wishlist

### Digital

| Game                                            | Why it matters                                                                                                                                   |
| ----------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Pushmo / Pullblox**                           | Core 3DS eShop loss. Great Nintendo puzzle game.                                                                                                 |
| **Crashmo / Fallblox**                          | Sequel with different rules; not redundant.                                                                                                      |
| **Stretchmo / Fullblox**                        | Third 3DS entry; free-to-start structure, but real game underneath.                                                                              |
| **BOXBOY!**                                     | HAL puzzle series opener.                                                                                                                        |
| **BOXBOXBOY!**                                  | Sequel.                                                                                                                                          |
| **BYE-BYE BOXBOY!**                             | Third entry.                                                                                                                                     |
| **Dillon’s Rolling Western**                    | Nintendo action/tower-defense hybrid.                                                                                                            |
| **Dillon’s Rolling Western: The Last Ranger**   | Sequel.                                                                                                                                          |
| **Dillon’s Dead-Heat Breakers**                 | Third entry; weird and more ambitious.                                                                                                           |
| **Attack of the Friday Monsters! A Tokyo Tale** | Level-5 Guild title; short, atmospheric, unique. Nintendo Life and Kotaku both highlight it among notable 3DS eShop losses. ([Nintendo Life][1]) |
| **Crimson Shroud**                              | Yasumi Matsuno tabletop-style RPG. One of the most important 3DS eShop exclusives. ([Kotaku][2])                                                 |
| **Weapon Shop de Omasse**                       | Rhythm/shop sim Guild title; weird in the good way.                                                                                              |
| **Liberation Maiden**                           | Suda51/Level-5 shooter.                                                                                                                          |
| **Aero Porter**                                 | Yoot Saito airport/luggage puzzle sim.                                                                                                           |
| **The Starship Damrey**                         | Guild horror/adventure oddity.                                                                                                                   |
| **Bugs vs. Tanks!**                             | Keiji Inafune Guild weirdness; not essential, but trapped.                                                                                       |

| Game                                                            | Notes                                                                                                                                   |

| **Kirby Fighters Deluxe**                                       | Expanded Kirby Triple Deluxe subgame.                                                                                                   |

| **My Nintendo Picross: The Legend of Zelda: Twilight Princess** | Reward-only Picross title; preservation curiosity.                                                                                      |
