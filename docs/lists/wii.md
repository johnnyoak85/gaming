# Nintendo Wii

## Wishlist

### WiiWare

| Game                                   | Why it matters                                                                                                                                 |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| **Castlevania: The Adventure ReBirth** | Digital-only Castlevania remake/reimagining. Still trapped.                                                                                    |
| **Contra ReBirth**                     | Same deal: Konami retro revival, never collected.                                                                                              |
| **Gradius ReBirth**                    | Completes the ReBirth trilogy.                                                                                                                 |
| **Doc Louis’s Punch-Out!!**            | Club Nintendo/WiiWare Punch-Out!! bonus fight; never re-released. Kotaku specifically notes it as one of the Club Nintendo WiiWare exclusives. |
| **Fluidity / Hydroventure**            | One of Nintendo’s best WiiWare originals; motion-controlled water puzzle platformer.                                                           |
| **FAST Racing League**                 | Predecessor to FAST Racing Neo / RMX, but distinct and still WiiWare-only.                                                                     |
| **You, Me, and the Cubes**             | Nintendo-published experimental physics puzzler. Very WiiWare.                                                                                 |
