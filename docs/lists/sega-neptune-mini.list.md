### On Mega Drive Mini 2

| Game                           | Why include it                                                                                                                                         |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Ecco the Dolphin CD**        | The enhanced CD version of one of Sega’s most recognizable atmospheric adventure games, with CD audio and presentation upgrades.                       |
| **Ecco: The Tides of Time CD** | Same logic as the first Ecco CD: a Sega-owned showcase for how Mega-CD could enhance an existing Mega Drive series.                                    |
| **Final Fight CD**             | One of the Mega-CD’s strongest arcade conversions, and arguably the most important home version of *Final Fight* before later arcade-perfect releases. |
| **Mansion of Hidden Souls**    | A distinctive first-person adventure/horror curiosity; very representative of the Mega-CD’s “CD-ROM weirdness” side.                                   |
| **Night Striker**              | A flashy Taito arcade rail-shooter conversion that fits the Mega-CD’s arcade-port identity well.                                                       |
| **Night Trap**                 | Historically infamous FMV title; not included because it is amazing, included because it is undeniably part of Sega CD history.                        |
| **The Ninja Warriors**         | Another Taito arcade conversion; a good example of Mega-CD being used for arcade-at-home presentation and music.                                       |
| **Robo Aleste**                | Compile shooter pedigree and one of the stronger original Mega-CD action games.                                                                        |
| **Sewer Shark**                | FMV-era historical texture. Like *Night Trap*, more important as a format/time-capsule piece than as a “best game.”                                    |
| **Shining Force CD**           | One of the actually important Mega-CD RPG/strategy titles, and a natural companion to the Mini 2’s *Shining Force II*.                                 |
| **Silpheed**                   | One of the iconic Mega-CD technical showpieces: polygonal/pre-rendered spectacle, very tied to the add-on’s identity.                                  |
| **Sonic CD**                   | The obvious mandatory one. Mainline-adjacent Sonic, major Sega CD identity title, and the system’s most famous game.                                   |

Total: 12 games

## Sega 32X

| Game                                  | Why include it                                                                                                                   |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **Virtua Racing Deluxe**              | Probably the best “this is why 32X exists” game. Better home Virtua Racing than the SVP Mega Drive cart, historically important. |
| **Knuckles’ Chaotix**                 | The Sonic-family flagship. Not perfect, but absolutely mandatory.                                                                |
| **Star Wars Arcade**                  | One of the defining early polygonal 32X showcase games.                                                                          |
| **Shadow Squadron / Stellar Assault** | Maybe the best original-ish 3D space combat game on the system. Very representative of the hardware.                             |
| **Kolibri**                           | Weird, beautiful, 32X-only, from the Ecco lineage. Exactly the kind of cult oddity a mini should preserve.                       |
| **Tempo**                             | Distinctive Sega platformer, very 32X, very “lost Sega mascot era.”                                                              |
| **After Burner Complete**             | Strong arcade conversion, and a better “arcade at home” argument than some of the weaker ports.                                  |
| **Space Harrier**                     | Historically right for the 32X as part of Sega’s arcade-at-home lineage.                                                         |
| **Virtua Fighter**                    | Compromised, yes. But historically important: Sega trying to bring Saturn-era 3D prestige to the add-on.                         |
| **Doom**                              | Not the best Doom, but historically important and very associated with the “can 32X do PC/next-gen stuff?” question.             |
| **Metal Head**                        | Another polygonal 3D showcase. Janky, but very 32X-coded.                                                                        |
| **Blackthorne**                       | Good game, but not 32X-essential because it exists elsewhere. Include only if you want quality over identity.                    |
| **Mortal Kombat II**                  | Good port and culturally relevant, but licensing-heavy and not uniquely 32X.                                                     |
| **NBA Jam Tournament Edition**        | Good and culturally relevant, but not 32X-defining.                                                                              |
| **WWF WrestleMania: The Arcade Game** | Good arcade-sports spectacle, but licensing-heavy.                                                                               |
| **Darxide**                           | PAL-exclusive rarity, technically interesting, very collector-relevant. Include if region/licensing is no issue.                 |

Total: 16 games

Grand Total: 35 games
