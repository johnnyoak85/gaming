I think we could use the rule of three circles.

- Major Genre: Shooter
- Minor Genre: Scroll Shooter
- Sub Genre: Precision Shooter

Although, we should not mix settings and themes with genres.
For example: Fantasy Shooter. Fantasy is the setting/theme, not the genre.

Please be very critical about this, I want to fully lock this down and create a new gaming taxonomy system that actually makes sense.

Cute should not be a theme. You immediatly ran into a problem but ignored it. Cotton's theme is "Fantasy". Cute, should be a Tag. Unless you can come up with some other bucket.
Likewise, I don't view Score as a genre, but perhaps it should be a tag. I don't view Score as a genre at all.

For example:
New Super Mario Bros. 2:

- Major: Platformer
- Minor: Adventure
- Sub: ??
- Tag: Money?

But let's explore Perspective, Pace and Structure. What could those be?

New property: Tone. Good. Let's explore this.

Alright, let's take a step back because this is becoming a little too granular which is similar to another system we once devised that I ended up abandoning. Let's establish context.

1. I am creating this system primarly for myself and for what I care for. If anyone else ends up using it, that's just a happy consequence.
2. The primary issue I am trying to solve is "If you liked this game, try this game".
    Examples:
    - If I search games like A Link to the Past, I don't want to see Breath of the Wild. I want to see Link's Awakening, Link Between Worlds, Eastward, Turnip Boy, etc.
    - If I search Tekken, I want to see Soul Calibur, DOA, Street Fighter IV. I do not want to see Street Fighter II.
    - If I search Twilight Princess, I might see Okami.
3. So I want the benefits of shit like "Metroidvania" without the stupid ass names. But that may become unavoidable.
4. I also want to search for specific stuff like "Games with Nobuo Uematsu soundtracks", "Igavania", "Games with designs by Akira Toriyama", without going too granular and end up with a credits page with people I don't really care about and will never search for.
